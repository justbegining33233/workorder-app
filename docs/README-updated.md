# FixTray - Work Order Software

## Overview
A full-featured work order management system for roadside and in-shop services. Built with Next.js, TypeScript, Tailwind CSS, and Prisma.

## Features
- Dashboard & work order listing
- Status filters and visual indicators
- Responsive design
- Work order management (CRUD)
- Priority levels and assignment
- RESTful API
- In-memory and PostgreSQL support
- Role-based access (admin, manager, tech, customer, shop)
- Security hardening (rate limiting, CORS, XSS, SQL injection protection)
- Comprehensive testing (unit, integration, E2E)
- Error handling & monitoring
- Docker & CI/CD
- Performance optimization
- Documentation & compliance
- Advanced admin features
- Analytics & reporting
